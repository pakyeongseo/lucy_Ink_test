# Lumi Ink v1.7 Release Notes

배포 기준: 2026-06-29  
내부 누적 기준: v66.46

## 주요 변경

- 정규식 작업실을 배포 기준 기능으로 정리했습니다. SillyTavern Regex용 IN/OUT 작성, 기본 문법 검증, 샘플 자동 생성, 캡처 미리보기, 샌드박스 미리보기, JSON 내보내기를 지원합니다.
- 정규식 작업실의 입력 동작을 다듬었습니다. 샘플만 있어도 IN 자동 생성이 가능하며, `[[값]]` 표기는 편집 중 보존하고 미리보기에서는 표식을 제거해 표시합니다.
- 정규식 옵션 명칭을 SillyTavern 기준에 맞춰 정리했습니다. placement, 비활성화됨, 편집 실행, Macros in Find Regex, Ephemerality 옵션을 내보내기 JSON에 반영합니다.
- 코드 작업실에서 HTML, JSON, Markdown 원문을 편집하고 미리볼 수 있습니다. HTML 미리보기는 샌드박스 iframe을 사용합니다.
- 자유 메모의 내 글 바로가기와 구분선 삽입 도구를 정리했습니다. 내 글 바로가기는 아이디어 보드형 버튼 디자인과 버튼 글자색 설정을 지원합니다.
- 메모 상단의 `기록 중`, `저장됨` 상태 표기와 왼쪽 상태 점 디자인을 복구했습니다.
- 퀵메뉴 슬롯 수 설정을 추가했습니다. 새 설치 기본값은 5개이며, 기존 사용자의 슬롯 값이 없던 경우에는 기존 7개 구성을 보존합니다.
- 퀵메뉴에 등록된 아이콘의 불필요한 외곽 라인을 제거했습니다.
- 아이디어 보드의 이미지, GIF, 음원, 영상, 파일, 메모 링크, HTML/PNG 내보내기 흐름을 현재 기준 문서에 반영했습니다.

## 배포 메모

- 서비스 워커 캐시는 `ink-memo-v1.7-release`로 갱신했습니다.
- 기존 브라우저에 설치된 PWA는 새 파일을 연 뒤 한 번 새로고침하면 새 캐시가 적용됩니다.
- 루미잉크는 로컬 우선 앱입니다. 배포판 교체 전후에는 전체 백업 파일을 별도로 저장하는 것을 권장합니다.
